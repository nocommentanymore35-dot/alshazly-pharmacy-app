# تصميم تطبيق صيدلية الشاذلي

## نظرة عامة
تطبيق موبايل أندرويد لصيدلية الشاذلي يتيح للعملاء تصفح الأدوية وطلبها مع لوحة إدارة مدمجة داخل التطبيق.

## قائمة الشاشات

### 1. الصفحة الرئيسية (Home)
- هيدر أزرق داكن يحتوي على اسم الصيدلية "صيدلية الشاذلي" والشعار "أدويتك الموثوقة في أيدٍ أمينة"
- شريط بحث أبيض مع أيقونة بحث ونص "ابحث عن دواء..."
- شريط إعلانات (Banner Carousel) يتبدل كل 7 ثواني تلقائياً مع نقاط تنقل
- قسم الفئات: أزرار أفقية قابلة للتمرير (مسكنات، فيتامينات، أدوية عامة، مضادات حيوية...)
- قائمة الأدوية: بطاقات عمودية تحتوي على صورة الدواء، الاسم (عربي/إنجليزي)، السعر بالجنيه المصري
- عند الضغط على دواء: شاشة تفاصيل الدواء مع خيار اختيار الكمية وإضافة للسلة أو المفضلة

### 2. السلة (Cart)
- قائمة الأدوية المضافة مع صورة وسعر وكمية (غير قابلة للتعديل)
- زر حذف لكل عنصر
- السعر الإجمالي في الأسفل
- زر "تأكيد الطلب" يفتح شاشة اختيار طريقة الدفع

### 3. المفضلة (Favorites)
- قائمة الأدوية المحفوظة كمفضلة
- زر إزالة من المفضلة
- زر إضافة للسلة مباشرة

### 4. الملف الشخصي (Profile)
- خانة الاسم الكامل
- خانة رقم الهاتف
- خانة العنوان التفصيلي (مكان التوصيل)
- زر حفظ البيانات
- قسم "طلباتي" لعرض حالات الطلبات السابقة

### 5. الإدارة (Admin) - محمية بكلمة مرور
- شاشة تسجيل دخول الإدارة (اسم مستخدم + كلمة مرور)
- إدارة الأدوية (إضافة، تعديل، حذف، رفع صور)
- إدارة الفئات
- إدارة الإعلانات (إضافة، تعديل، حذف، ترتيب)
- إدارة الطلبات (عرض الطلبات، تغيير الحالة)
- التقارير (مبيعات وإيرادات)

## تدفقات المستخدم الرئيسية

### تدفق الطلب:
الرئيسية → اختيار دواء → تحديد الكمية → إضافة للسلة → السلة → تأكيد الطلب → اختيار الدفع (نقدي/فودافون كاش) → تأكيد → إشعار للإدارة

### تدفق تتبع الطلب:
الملف الشخصي → طلباتي → عرض حالة الطلب (استقبال → تجهيز → شحن → تسليم)

### تدفق الإدارة:
تبويب الإدارة → تسجيل الدخول → لوحة التحكم → إدارة الأدوية/الإعلانات/الطلبات/التقارير

## شريط التنقل السفلي (من اليمين لليسار)
1. الرئيسية (أيقونة منزل)
2. السلة (أيقونة سهم/إرسال)
3. المفضلة (أيقونة قلب)
4. الملف الشخصي (أيقونة أقواس)
5. الإدارة (أيقونة ترس)

## طرق الدفع
1. الدفع عند الاستلام
2. فودافون كاش (تفاصيل تُضاف لاحقاً)

## حالات الطلب
1. تم استقبال الطلب
2. قيد التجهيز
3. تم الشحن
4. تم التسليم
